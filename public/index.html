    📘 Guía rápida de opciones

    Formatos de salida
    • WEBP: buena calidad/peso, muy compatible en web.
    • AVIF: compresión excelente; puede ser más lento de codificar.
    • JPEG: clásico con pérdidas, ideal para fotos.
    • PNG: sin pérdidas; perfecto para gráficos/íconos/transparencias.
    • TIFF: uso profesional/archivo; archivos más pesados.

    ✔️ Puedes marcar varios; cada imagen se convierte a todos los formatos seleccionados.

    Quality (1–100)

    Controla la calidad visual (principalmente en WEBP/AVIF/JPEG):

    • 100 = máxima calidad, archivos grandes.
    • 1 = mínima calidad, archivos muy pequeños.
    • Recomendado: 70–85 para buen balance.
    Effort (0–9)

    Define cuánto “esfuerzo” pone el compresor (más esfuerzo = mejor compresión pero más lento):

    • 0: muy rápido, menos eficiente.
    • 9: muy lento, máxima compresión.
    • WEBP usa 0–6; AVIF usa 0–9.
    WebP Lossless

    Convierte a WebP sin pérdidas (la imagen queda idéntica al original). Los archivos pueden ser más grandes.

    JPEG (mozjpeg)

    Usa un codificador optimizado para JPEG que mejora la compresión y los artefactos a igual calidad.

    PNG compressionLevel (0–9)
    • 0: sin compresión (rápido; archivos grandes).
    • 9: máxima compresión (lento; archivos pequeños).
    • La calidad en PNG es siempre sin pérdidas.
    Consejos
    • Para web: prueba WEBP con quality 75–85 y effort 4.
    • Para máxima compresión: AVIF con quality 45–60, effort 6–8.
    • Para logos/íconos: PNG (compressionLevel 6–9) o WEBP lossless.
