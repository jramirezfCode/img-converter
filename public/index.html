Formatos de salida

WEBP — Ideal para web, buen peso/calidad.
AVIF — Mejor compresión, más lento de codificar.
JPEG — Fotos con pérdidas, clásico y compatible.
PNG — Sin pérdidas, ideal para transparencias.
TIFF — Uso profesional y archivo.

Calidad y esfuerzo

Calidad 70–85 es el punto óptimo para web. Mayor esfuerzo produce archivos más pequeños pero tarda más.

Redimensionar (ancho y alto)

Solo indicas el ancho máximo. El alto se calcula solo para mantener la proporción original — no se deforma la imagen. Una foto 3000×2000 (ratio 3:2) se reescala a 1200×800. Si la imagen ya es más pequeña que el ancho indicado, se deja sin cambios. Déjalo apagado si ya pre-procesaste las fotos en Canva, Figma o Photoshop.

Renombrado en lote

Con prefijo clinica-pilar e inicio 1, obtienes clinica-pilar-01.webp, clinica-pilar-02.webp, etc. Si el toggle está apagado, se mantienen los nombres originales.

Eliminar metadatos

Quita EXIF, GPS, creador, software y perfil ICC. Reduce el peso del archivo y protege tu privacidad al compartir.

Nombre del archivo ZIP

Cada descarga incluye fecha y hora en el nombre, por ejemplo convertidos_20260423_114305.zip, para que nunca se sobreescriban ni se confundan entre lotes.
